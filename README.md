# Network Port Scanner

## Description
Python based TCP port scanner developed in kali Linux using socket programming to identify open ports and demonstarte network security fundamentals.

## Features
- Scan ports from 1 to 9000
- Detect open ports
- Display total number of open ports
- Simple command-line interface

## Technologies Used
- Python 3
- Socket Programming
- Kali Linux
- Target IP Input
- Open Port Detection
- Service Detection(Http,ssh,etc)
- Scan Duration Measurement
- Report Generation
- Cyber Security
- Network Security
- Ethical Hacking

## How to Run

```bash
python3 port_scanner.py
