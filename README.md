# Network Port Scanner

## Description
A Python-based TCP Port Scanner developed in Kali Linux.

The scanner checks a target host for open ports and displays the results.

## Features
- Scan ports from 1 to 9000
- Detect open ports
- Display total number of open ports
- Simple command-line interface

## Technologies Used
- Python 3
- Socket Programming
- Kali Linux

## How to Run

```bash
python3 port_scanner.py
